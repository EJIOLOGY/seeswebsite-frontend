# SEES Website

# Getting Started

Navigate to directory on your machine:

```bash
cd folder/to/clone-into/
```

Clone repository:

```bash
https://github.com/SEEES-TECH-COMMUNITY/seeswebsite-frontend.git
```

Install all dependencies:

```bash
pnpm install
```


First, run the development server:

```bash
pnpm dev
```
